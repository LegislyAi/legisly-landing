# Legisly Design System

For all frontend and UI work, follow the design system defined here.

## Color Palette

| Variable | Hex | Usage |
|----------|-----|-------|
| --blue-primary | #1E40AF | Primary brand color, headings, feature card borders |
| --blue-deep | #0B1D5E | Dark backgrounds, text shadows, borders |
| --blue-bright | #2563EB | Buttons, links, accent highlights |
| --blue-electric | #3B82F6 | Hover states, secondary accents |
| --orange-hot | #FF6B35 | Primary accent, CTA buttons, feature borders |
| --orange-bright | #FF8C42 | Hover states, secondary orange |
| --yellow-bright | #FFD60A | Primary CTA button fill, logo color, stat numbers |
| --yellow-warm | #FDE047 | Emphasis text, secondary yellow |
| --coral | #FF4D6D | Text shadows, badge backgrounds, energy accent |
| --cream | #FFFDF5 | Light section backgrounds |
| --white | #FFFFFF | Card backgrounds |

## Typography

- Headlines / Logo: Archivo Black (400)
- Logo wordmark: Righteous
- Body / UI: Space Grotesk (400, 500, 600, 700)

## Key Rules

- Never use border-radius on cards or buttons
- All borders 3-4px solid minimum
- Box shadows use brand accent colors, not gray
- Text-transform: uppercase on all headlines, buttons, labels
- Card shadow color rotation: blue-bright, orange-hot, coral, yellow-bright
